# K线图查询页面使用说明

## 功能介绍

主页现在是一个专业的股票K线图查询页面，支持：

- 📊 实时K线图展示（日线、周线、月线）
- 📈 多条移动平均线（MA5、MA10、MA20、MA30）
- 📉 成交量柱状图
- 🔍 自定义日期范围查询
- 💹 实时股票信息展示（最新价、涨跌幅等）
- 🎯 数据缩放和拖拽功能

## 启动步骤

### 1. 启动后端服务

```bash
cd backend
python app.py
```

后端将在 http://localhost:5000 运行

### 2. 启动前端服务

```bash
cd frontend
npm run dev
```

前端将在 http://localhost:5173 运行

## 使用方法

1. **输入股票代码**：例如 `000001.SZ`（平安银行）
2. **选择周期**：日线(D)、周线(W)、月线(M)
3. **选择日期范围**：开始日期和结束日期（格式：YYYYMMDD）
4. **点击查询**：系统将获取K线数据并展示图表

## 常用股票代码示例

- `000001.SZ` - 平安银行
- `600000.SH` - 浦发银行
- `000002.SZ` - 万科A
- `600519.SH` - 贵州茅台

## 功能说明

### 股票信息栏
显示当前股票的实时信息：
- 股票名称
- 最新价格
- 涨跌额（红涨绿跌）
- 涨跌幅
- 最高价/最低价
- 成交量

### K线图表
- **蜡烛图**：红色上涨，绿色下跌
- **移动平均线**：MA5、MA10、MA20、MA30
- **成交量**：底部柱状图，颜色对应涨跌
- **缩放功能**：鼠标滚轮缩放，拖拽查看不同时间段
- **提示框**：鼠标悬停显示详细数据

## 技术栈

- **前端**：Vue 3 + Element Plus + ECharts
- **后端**：Flask + Tushare
- **数据源**：Tushare金融数据接口
