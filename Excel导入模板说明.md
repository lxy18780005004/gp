# Excel导入模板说明

## Excel文件格式要求

### 文件格式
- 支持 `.xlsx` 和 `.xls` 格式
- 股票代码应放在第一列（A列）
- 从第一行开始读取

### 示例格式

| A列（股票代码） |
|----------------|
| 000001.SZ      |
| 600000.SH      |
| 000002.SZ      |
| 600519.SH      |
| 000858.SZ      |

### 使用步骤

1. **准备Excel文件**
   - 在Excel的A列输入股票代码
   - 每行一个代码
   - 保存为 `.xlsx` 或 `.xls` 格式

2. **导入文件**
   - 点击"导入Excel"按钮
   - 选择准备好的Excel文件
   - 系统会自动读取所有股票代码并批量查询

3. **查看结果**
   - 图表会以2列网格形式展示
   - 每个图表显示股票名称、代码、最新价和涨跌幅
   - 支持独立缩放和查看每个图表

## 功能特点

### 批量查询
- 支持手动输入多个股票代码（用逗号分隔）
- 支持Excel文件批量导入
- 并发请求，快速加载

### 紧凑布局
- 一行显示2个图表
- 图表高度优化为400px
- 减少内边距，节约空间
- 响应式设计，小屏幕自动切换为单列

### 图表功能
- K线图 + 成交量
- MA5、MA10、MA20、MA30均线
- 数据缩放和拖拽
- 鼠标悬停显示详细信息

## 注意事项

1. 股票代码格式必须正确（如：000001.SZ 或 600000.SH）
2. 后端服务必须启动（http://localhost:5000）
3. 批量查询时建议不超过20个股票，避免请求过多
4. Excel文件中的空行会被自动忽略
